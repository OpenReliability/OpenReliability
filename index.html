OpenReliability

A free microelectronics reliability data analysis software

View on GitHub Download .zip Download .tar.gz

Reliability analysis made easy

OpenReliability aims to develop a free and open-source data analysis software for microelectronics reliability engineers.

OpenReliability provides an easy to use and consistent interface to analyze reliability experiment (electromigration, TTDB, HCI, NBTI...) and generate professionally looking charts (cumulative failure distribution, wafer maps...).

Languages

OpenReliability is developed in Python and R.

License

OpenReliability is released under GPL version 2.

Version

OpenReliability is currently at a very early alpha stage. But stay tuned, it's coming soon. ;)
